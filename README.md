# ParOuImpar
A construção do jogo "Par Ou Impar", tem como objetivo colocar em prática os conhecimentos em javascript, funções e Vue.js.
Trabalho avaliativo da matéria de Códigos de Alta Performance Web. O jogo funciona da seguinte maneira, o jogador escolhe par ou impar e digita o número desejado, ao clicar no botão "escolha" a maquina irá sortear um número de um a cinco. Em seguida, ao "jogar", o Javascript será responsável por fazer a soma e verificação se o jogador ganhou ou perdeu de acordo com sua escolha.
